Features

Everything your users need.
Nothing in the way.

Built from the ground up for developer-focused projects. Your guide is always there, never annoying, and always on topic.

🧠

Context-Aware

The guide reads the page and forms its own understanding. No scripting required — it knows what it's looking at.

⚡

Instant Integration

One script tag. Any framework. Zero configuration to get started and sensible defaults for every setting.

🎨

Fully Themeable

Swap the guide character, voice, colors, and personality to match your brand without touching core logic.

🔒

Privacy First

Zero tracking, no third-party calls, and all data stays local. Works completely offline after the first load.

♿

Accessible by Default

Built to WCAG 2.1 AA. Screen reader friendly, keyboard navigable, and respects prefers-reduced-motion.

🌍

Multilingual

Ships with support for 28 languages out of the box. Add your own locale file in minutes.

Quick Start

Running in under
a minute.

Install via npm, add one line to your HTML, and your guide is live. No build step needed for basic usage.

terminal 

# 1 — Install
npm install livingdocs

# 2 — Add to your page
<script src="livingdocs.min.js"></script>

# 3 — Initialize (optional config)
LivingDocs.init({ character: 'face.png', theme: 'cyber' });

About

Why living docs?

Static documentation fails the moment someone lands on the wrong page, doesn't know the terminology, or just needs a nudge in the right direction. A guide that watches, reacts, and responds changes that completely.

Started as a weekend project, it's now used by developer tools, SaaS products, and internal knowledge bases across 40+ countries.
